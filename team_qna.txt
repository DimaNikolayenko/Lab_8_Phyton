Ланцюжок питань і відповідей
------------------------------------
Студент: Ніколаєнко Дмитро
Питання: Як у Python працювати з рядками: знайти довжину, змінити регістр, порахувати кількість певного символу?
------------------------------------
Студент: Пушкар Юлія
Відповідь: У Python з рядками зручно працювати за допомогою вбудованих методів. Функція len() визначає довжину тексту, а методи upper() і lower() змінюють регістр символів. Для підрахунку кількості певного символу використовують метод count('a').
Питання: Як створити словник у Python, додати в нього нову пару ключ-значення та вивести всі елементи?
------------------------------------
Студент: Ганжара Богдан
Відповідь: Словник у Python створюється за допомогою фігурних дужок: d = {}. Нову пару додають командою d['ключ'] = 'значення', а всі елементи можна переглянути циклом for key, value in d.items():.
Питання: Як створити власну функцію користувача у Python, яка приймає аргументи і повертає результат?
------------------------------------
Студент: Писана Ольга
Відповідь: Власну функцію створюють за допомогою ключового слова def. Вона може приймати параметри й повертати результат через оператор return. Наприклад: def add(a, b): return a + b
